StudyMind — AI Study Planner

A beautiful, feature-rich AI-inspired study planner built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

StudyMind is a modern, dark-themed productivity web app designed to help students organize their study sessions, track progress across subjects, manage tasks efficiently, and stay motivated throughout their academic journey. Everything runs in the browser with data persisted via LocalStorage — no backend, no login required.
